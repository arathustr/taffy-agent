# 产品规格

## 产品愿景

Taffy Agent 是一个“桌宠形态的本地操作 Agent 终端”。它把虚拟主播主题的陪伴感、桌面助手的即时性、Codex 的编程能力、浏览器自动化的网络操作能力合并在一起。

最终体验应该像这样：

1. 用户用自然语言对桌面上的塔菲说任务。
2. 塔菲理解意图，判断是否需要浏览器、Codex、Shell、文件系统或用户确认。
3. 塔菲用动态表情和任务流展示正在做什么。
4. 遇到登录、验证码、风险操作时，塔菲暂停并请求用户接管。
5. 任务完成后，塔菲给出结果、变更摘要、后续建议和可复盘记录。

## 目标用户

- 主要用户：熟悉电脑操作、希望用 Agent 辅助写代码和做项目的塔菲粉丝。
- 使用场景：本地开发、浏览器检索、网页操作、项目创建、调用 Codex 修 bug、让桌宠汇报进展。
- 使用环境：Windows 桌面优先，后续可扩展到 macOS/Linux。

## 产品支柱

### 1. 有存在感的桌宠

- 透明窗口、置顶、可拖动、可缩放。
- 空闲、倾听、思考、执行、等待确认、完成、出错等动态状态。
- 支持 Live2D 优先，GIF/PNG 作为早期占位。
- 任务气泡和小型状态条跟随角色，而不是把用户拉进复杂控制台。

### 2. 逼真的角色交互

- 语言风格要有角色感，但不牺牲清晰度。
- 不把每句话都写成段子；工作状态下要短、准、能执行。
- 支持文本、语音、点击、拖拽、快捷键、右键菜单和命令面板。
- 支持打断、追问、澄清、复述计划、确认危险动作。

### 3. 强 Agent 能力

- 意图识别、任务拆解、工具选择、执行、复盘。
- 支持同步小任务和异步长任务。
- 支持 Planner、Executor、Reviewer、Memory、Policy Gate。
- 工具调用采用结构化协议，不依赖散文式输出驱动业务逻辑。

### 4. 高适配 Codex

- 检测本机 Codex CLI/桌面配置是否可用。
- 帮用户整理 Codex 任务说明、启动 Codex、观察项目变更和终端输出。
- Codex 账号、模型、密钥、额度由 Codex 自己管理，Taffy 不接管。
- Taffy 可以作为“项目经理”和“翻译器”，让用户用更轻松的话把任务交给 Codex。

### 5. 丝滑浏览器操作

- 能搜索、打开网页、读页面、点击、输入、截图、下载、等待 DOM 状态。
- 登录流程进入受控模式：用户输入账号密码/2FA，Taffy 观察登录是否完成后继续。
- 遇到验证码、风控、支付、账号设置等敏感动作，必须暂停并让用户确认。

## 完整版功能范围

| 模块 | 功能 |
| --- | --- |
| 桌宠 UI | Live2D/GIF 动画、透明窗口、气泡、任务时间线、模式切换 |
| 对话 | 文本/语音输入、TTS、字幕、打断、澄清、上下文记忆 |
| Agent Core | 路由、规划、工具调用、复盘、失败恢复、长任务状态 |
| CodexBridge | 启动/派发 Codex 任务、读取状态、总结 diff/test/log |
| BrowserBridge | 自动浏览、登录接管、表单、下载、截图、页面摘要 |
| ShellBridge | 创建项目、运行脚本、安装依赖、测试、打包 |
| FileBridge | 文件读写、目录摘要、项目索引、变更预览 |
| Memory | 用户偏好、项目记忆、任务历史、可删除记忆 |
| Config | DeepSeek Key、模型选择、代理、权限、声音、角色选项 |
| Observability | JSONL 日志、工具 trace、失败原因、可复盘任务记录 |

## 非目标

- 不做 Codex 账号代理，不保存 Codex 凭据。
- 不绕过验证码、2FA、风控或网站限制。
- 不默认保存用户网站密码。
- 不做面向商业分发的塔菲官方形象授权包。
- 不承诺全自动处理所有网页登录；目标是“顺滑的人类接管+自动继续”。
